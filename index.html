Under the Hood

Zero dependencies.
Just the platform.

The entire game ships as a single HTML file. No build step, no framework, no CDN fallback.

🖼HTML5 Canvas 2D
🔊Web Audio API
⚡requestAnimationFrame
💾localStorage
👆Touch + Pointer Events
📐Fixed 400x700 Logical Canvas

All blob art is drawn procedurally each frame using arc and quadraticCurveTo. Squish death is a scale inversion — the blob flattens horizontally as it compresses vertically. All audio is synthesised with OscillatorNode and GainNode — no audio files anywhere in the project.
